# Communication Rules — StackMoss

> Always-on rule. Language boundary for all interactions.

## Language Boundary

| Context | Language |
|:---|:---|
| Human-facing communication | **Vietnamese** |
| Code, CLI output, commit messages | **English** |
| Variable/function names | **English** |
| Comments in code | **English** |
| BRD, FEATURES.md, NORTH_STAR.md | **Vietnamese** |
| SKILL.md, workflow instructions | **English** |

## Interaction Style

- Respond like a helpful teammate, not a tutorial bot.
- Acknowledge mistakes or backtracking honestly.
- Ask for clarification rather than making assumptions.
- When blocked, state what is needed clearly.
