## Claude Code Bootstrap

This repository uses Claude Code project memory via `CLAUDE.md`.
Shared cross-agent guidance is imported from `AGENTS.md`.

## Skills

- Project skills live in `.claude/skills/<skill-name>/SKILL.md`.
- `skill-creator` is available at `.claude/skills/skill-creator/SKILL.md`.

## Skill Creator Usage

- Invoke with `/skill-creator` or let Claude auto-trigger when the request is about creating/upgrading skills.
- For technical skills generated by `skill-creator`, require executable validation and failure logging.
- If validation cannot run due to missing prerequisites, ask focused owner questions before marking done.
